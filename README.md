# Greetings! Welcome to my motion planning repository!

## Include motion planner:
1. BEAST
2. B-SST
3. ATEMPTS (working on)

## Usage:
```
mkdir build
cd build
cmake ..
make
./MotionPlanning problemSetting.inst
```

## OMPL:
To compile the program, you need to install OMPL first:
